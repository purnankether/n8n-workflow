# RAG Workflow for n8n

A Retrieval-Augmented Generation workflow that lets you chat with your documents.

## Features
- Automatic document ingestion and indexing
- AI-powered chat interface with context retrieval
- Conversation memory for follow-up questions

## Prerequisites
- n8n installed (self-hosted or cloud)
- OpenAI API key

## Installation
1. Import `RAGworkflow.json` into n8n
2. Configure OpenAI credentials
3. Replace sample documents with your content
4. Run the ingestion trigger
